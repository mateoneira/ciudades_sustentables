# Urban Data Science
Data Science course for Sustainable Cities MSc at UCuenca

**Language**: Spanish

**Format**: 

* markdown files for lectures. the markdown files will be used to create powerpoint slides for the lectures. 
* Juputer notebooks for tutorials and assignments. 

---
### Información General

* **Objetivos:** La asignatura tiene por objetivo desarrollar destrezas en el manejo de análisis de datos para aplicaciones en **sostenibilidad urbana**.
* **Duración:** **60 horas** de docencia + **5 horas** de tutorial.

**Resultados de aprendizaje:**

* Conocer los fundamentos teóricos y prácticos del análisis de datos.
* Desarrollar destrezas en el análisis **exploratorio, inferencial y predictivo**.
* Conocer las nuevas tendencias en minería de datos y machine learning.

**Competencias a desarrollar:**

* Utiliza estructuras básicas de Python y sus librerías principales para importar, limpiar y organizar datos en estudios de sostenibilidad urbana.
* Aplica técnicas de análisis exploratorio de datos para identificar y visualizar patrones, tendencias y relaciones en problemáticas urbanas.
* Implementa métodos estadísticos e inferenciales para formular hipótesis y obtener conclusiones válidas en el análisis de datos.
* Explica y aplica los fundamentos de los modelos predictivos, identificando las fortalezas y limitaciones de algoritmos básicos de machine learning en contextos urbanos.

**Resumen del contenido**

* Análisis exploratorio de datos (EDA).
* Análisis inferencial.
* Modelos predictivos.
* Fundamentos de minería de datos y algoritmos de Machine Learning.
